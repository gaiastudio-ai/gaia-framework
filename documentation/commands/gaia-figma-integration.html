/figma-integration

internal
Category:
Internal Skills

What it is

OSS stub for the premium Figma integration skill. The full capability (design token extraction, component spec generation, frame authoring, asset export, import flow, and fidelity gate) ships in the enterprise plugin behind the figma-premium feature flag.

Called by

  • /gaia-create-ux (Figma hook)
  • /gaia-review-code (fidelity gate)

What it does

In the OSS version, this skill is a stub that documents the boundary and points to the enterprise activation path.

When the enterprise plugin is installed and the figma-premium feature flag is enabled, the full skill replaces this stub: design-tool MCP detection, design-token extraction, component specification, frame authoring, asset export, and design-to-implementation fidelity gate.

Without the enterprise plugin, consuming workflows continue in markdown-only mode.

When you will see it surface

  • If you invoke a UX design workflow and see 'Figma integration: OSS stub -- enterprise plugin required', this skill is the source.
  • The stub is always loadable without error -- it degrades gracefully to a redirect message.

Technical notes

  • Requires the gaia-enterprise plugin and figma-premium feature flag for full functionality.
  • Install via: /plugin marketplace add gaiastudio-ai/gaia-enterprise.
  • Without enterprise: zero Figma features, zero errors -- markdown-only operation.
