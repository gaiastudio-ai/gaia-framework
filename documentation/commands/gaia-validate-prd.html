What it does

Deprecated

This command is deprecated. It redirects to /gaia-val-validate, which handles all artifact validation. Use /gaia-val-validate .gaia/artifacts/planning-artifacts/prd.md directly.

/gaia-validate-prd was the original PRD validation command. Its validation logic has been consolidated into the general-purpose /gaia-val-validate command. This entry point is preserved for backward compatibility -- invoking it silently redirects to the canonical validator.

When to use it

Use /gaia-val-validate instead.

How to invoke

/gaia-validate-prd

Redirects to: /gaia-val-validate .gaia/artifacts/planning-artifacts/prd.md

Related commands

CommandRelationship
/gaia-val-validate The canonical replacement. Validates any artifact, including PRDs.
/gaia-create-prd Creates the PRD. Includes built-in validation.
/gaia-edit-prd If validation fails, edit the PRD to address findings.
